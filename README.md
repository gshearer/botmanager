## Introduction

BotManager - A modular, heavily threaded, fault tolerant framework for various BOTs written in C.

## Description

BotManager aims to provide a modular, heavily threaded, fault tolerant framework for various types of BOTs. A bot can be any automated process to perform functions such as scraping data from websites and/or APIs, interacting with humans via interfaces like IRC / XMPP / Slack / Telegram / HTTP, performing asset trading on behalf of humans, etc.

The framework provides database connectivity, network socket i/o with tls support, libcurl access, and IRC protocol support as "core services". All other functionality is provided by plugins.
